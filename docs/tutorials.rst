Tutorials
=========

This section provides comprehensive tutorials demonstrating how to use **SCILD**
and reproduce the analyses presented in the manuscript.

Each tutorial is provided as a Jupyter notebook that can be downloaded and executed locally.
The datasets used are publicly available and referenced within each notebook (https://doi.org/10.5281/zenodo.17522268).
