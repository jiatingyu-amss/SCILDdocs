Installation
============

SCILD requires Python 3.10+.

We recommend installing SCILD in a new conda environment:

.. code-block:: bash

   conda create -n scild python=3.10
   conda activate scild

Clone the repository and install dependencies:

.. code-block:: bash

   git clone https://github.com/jiatingyu-amss/SCILD.git
   cd SCILD
   pip install -r requirements.txt

Dependencies
------------

Main dependencies include:

- scanpy
- scipy
- pytorch
- commot

You can view the full list in the main repository's ``requirements.txt``.
